## gitbook

新版本使用`vitepres`构建，提供[预览模式](https://tangxiangmin.github.io/interview/index.html)

```
git clone git@github.com:tangxiangmin/interview.git
cd interview

npm i 

# vitepress开发和打包
npm run dev
npm run build
npm run serve
```
