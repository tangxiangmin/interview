## vitepress

新版本使用`vitepres`构建，提供[预览模式](https://tangxiangmin.github.io/interview/index.html)

```
git clone git@github.com:tangxiangmin/interview.git
cd interview

yarn

# 生成summary.md目录，使用之前旧版本的gitbook格式
yarn catalogue:build  

# vitepress开发和打包
yarn docs:dev
yarn docs:build
yarn docs:serve
```
