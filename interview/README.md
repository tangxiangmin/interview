前端面试
====

日常收集整理的的前端知识点和面试题。

面试的信心来自于自身扎实的基础，切勿急功近利，好逸恶劳~自勉之。

## gitbook
项目提供了[gitbook预览模式](https://tangxiangmin.github.io/interview/index.html)
```

git clone git@github.com:tangxiangmin/interview.git
cd interview

# run once, auto compile SUMMARY.md,
npm run setup

npm run build

gitbook serve
```
